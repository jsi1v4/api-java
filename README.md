- [Spring Boot](https://start.spring.io)

## Como baixar e executar o projeto

### Clonar o repo

```sh
git clone https://github.com/jsi1v4/api-java
```

### Executar em localhost

```sh
# run in http://localhost:5000
./gradlew bootRun
```

### Buildar projeto

```sh
./gradlew build
```
